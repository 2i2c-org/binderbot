# Contributing

BinderBot is maintained by [2i2c](https://2i2c.org) and developed at [2i2c-org/binderbot](https://github.com/2i2c-org/binderbot).
Contributions of all kinds are welcome.
See [](./about.md) for broader plans around the project.

- **Questions and bug reports**: open an issue at [github.com/2i2c-org/binderbot/issues](https://github.com/2i2c-org/binderbot/issues).
- **Code and docs**: pull requests are welcome; for larger changes, open an issue first to discuss the idea.

The [developer documentation](./contributing-guide.md) covers how to contribute and how the project is built, including architecture, development setup, and releases.
